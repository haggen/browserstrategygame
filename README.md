## Development

### Stack

- Python 3.12
- Poetry
- Ruff
- FastAPI
- SQLModel

### Getting started

Create virtual environment and install dependencies:

```shell
python -m venv .venv
source .venv/bin/activate
poetry env use python
poetry install
```

Start the application:

```shell
poetry run start
```
